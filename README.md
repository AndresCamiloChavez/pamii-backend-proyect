# Pamii API

1. ```yarn install```
2. Clonar el archivo ```.env.template``` y renombrar a ```env```

3. Cambiar las variables de entorno
4. Levantar la base de datos
```docker-compose up```
5. Levantar en modo de desarrollo ```yarn start:dev```
